# :parrot: COCKATIEL (ACL 2023)

*COCKATIEL: COntinuous Concept ranKed ATtribution with Interpretable ELements for explaining neural net classifiers on NLP*, Fanny Jourdan*, Agustin Picard*, Thomas Fel, Laurent Risser, Jean Michel Loubes & Nicholas Asher. Findings of ACL 2023, [[arXiv]](https://arxiv.org/abs/2305.06754).

The code is implemented and available **for Pytorch**. A notebook is available: [notebook example](./example.ipynb).
